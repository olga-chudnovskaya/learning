Базовые элементы JS

Создаем переменную d и кладем в нее объект со свойствами name (значение "James") и lastname (значение "Bond"):

            let d = {
                name: 'James',
                lastname: 'Bond'
            };
            // если подать команду в консоль "alert(d);", выскочит модальное окно со строкой '[object Object]'
            // если подать команду в консоль "alert(d.name);", выскочит модальное окно со строкой 'James'
            // если подать команду в консоль "alert(d.lastname);", выскочит модальное окно со строкой 'Bond'
            // если подать команду в консоль "alert(typeof d);", выскочит модальное окно со строкой "object"

Создаем переменную e. Присваиваем ей значение переменной d:

            let e;
            e = d;
            // убеждаемся, что в обеих переменных лежит один и тот же объект:
            // меняем одно из свойств объекта в переменной e -
            // и видим, что в переменной d оно тоже изменилось.

Создаем переменную f. Присваиваем ей пустой объект. Копируем в нее все свойства переменной d перебором в цикле:

            let f;
            f = {};
            for (let i in d) {f[i] = d[i]}
            // убеждаемся, что в обеих переменных лежат разные объекты:
            // меняем одно из свойств объекта в переменной d -
            // и видим, что в переменной f оно не изменилось.
